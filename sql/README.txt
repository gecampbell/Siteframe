Siteframe SQL Directory
$Id: README.txt,v 1.1 2003/06/27 05:00:10 glen Exp $

The principal files you will find in this directory:

siteframe.sql - the core data definitions for the current release
of Siteframe.

upgradeXXXXX.sql - upgrades the database from one version of Siteframe
to another.

templates.sql - defines all the standard themes and templates for
the current version of Siteframe.

templatesXXX.sql - themes and templates for prior versions.

object_properties.sql - adds additional properties to the base
document objects.
